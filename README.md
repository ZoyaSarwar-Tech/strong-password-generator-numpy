# 🔐 Strong Password Generator using NumPy

A simple yet powerful **Python project** that generates strong and secure passwords using NumPy.  
It ensures that every password contains a mix of uppercase letters, lowercase letters, digits, and special symbols for better security.

## 🚀 Features

- Generates strong random passwords
- Ensures at least:
  - 1 Uppercase letter
  - 1 Lowercase letter
  - 1 Digit
  - 1 Special symbol
- Custom password length support
- Fully random and secure output
- Beginner-friendly NumPy usage

## 🛠️ Technologies Used

- Python 🐍
- NumPy 🔢

## 📌 How It Works

1. User enters desired password length
2. Program ensures minimum security rules
3. Random characters are selected from:
   - lowercase letters
   - uppercase letters
   - digits
   - symbols
4. Characters are shuffled for randomness
5. Final secure password is generated

## ▶️ How to Run

```bash
python password_generator.py
